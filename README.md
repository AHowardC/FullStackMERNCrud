# FullStackMERNCrud

##

Technology Used:

<li>React</li>
<li>Redux</li>
<li>Redux-Thunk</li>
<li>Bootstrap</li>
<li>Express</li>
<li>Node</li>
<li>MongoDB</li>

##

Prerequisites :

<ol>
<li>Clone this repository</li>
<li>Install server dependencies</li>
<pre><code>$ cd server
$ npm install
</code></pre>
<li>Install client dependencies</li>
<pre><code>$ cd client
$ npm install
</code></pre>
</ol>

##

Run the app :

<ol>
<li>Set up MongoDB</li>
<li>Start the server</li>
<pre><code>$ cd server
$ npm start
</code></pre>
<li>Start the client</li>
<pre><code>$ cd client
$ npm start
</code></pre>
<li>Navigate to http://localhost:3000/</li>
</ol>
